# Обзор

Aki - это модульный бот, в котором большинство функций и команд могут быть включены/отключены.

**Набор модулей по умолчанию включает:**
 - Модерация (Скоро)
 - Роле-плей (Скоро)
 - Веселье
 - Утилиты
 - Музыка (Скоро)

В дополнение к этим модулям (когам), вы можете добавлять свои модули.
